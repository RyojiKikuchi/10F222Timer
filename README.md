# 10F222Timer

## プロジェクトの概要

PIC10F222 を使用した 1〜5 分計測タイマーです。  
ボリュームで計測時間を設定し、スイッチで計測を開始します。  
計測中は LED が点滅し、タイマー完了時にブザーで音楽を鳴らして通知します。

## ピンアサイン

| ピン | I/O | 出力形式 | 説明 |
|------|-----|----------|------|
| GP0 | 入力 (ADC) | アナログ入力 | ボリューム（可変抵抗）の電圧をADC読み取りしてタイマー時間を決定する |
| GP1 | 出力 | プッシュプル | ブザー（電圧スピーカ）へのトーン信号を出力する |
| GP2 | 出力 | プッシュプル | LED の点灯・点滅制御 |
| GP3 | 入力 | プルアップ | タクトスイッチ入力。プルアップ有効のため、押下時 LOW となる |

> **備考**  
> - MCLRE = 0 に設定し、GP3 を汎用 I/O（スイッチ入力）として使用しています。  
> - OPTION レジスタにより GP0・GP1・GP3 のプルアップを有効化しています。  
> - GP0 は ADCON0 の ANS0 ビットでアナログ入力として動作します。

## 回路接続

```
VDD (5V)
 │
 ├─[10kΩ プルアップ]─── GP3 ─── タクトスイッチ ─── GND
 │
 ├─[可変抵抗 10kΩ]─── GP0（ADC入力）
 │         ↑ 両端を VDD / GND に接続し、ワイパーを GP0 へ
 │
 ├─ GP2 ─[電流制限抵抗 330Ω]─── LED ─── GND
 │
 └─ GP1 ─── 電圧スピーカ（ブザー）─── GND
```

| 接続先 | ピン | 補足 |
|--------|------|------|
| ボリューム（可変抵抗） | GP0 | 10 kΩ 可変抵抗のワイパーを接続。両端は VDD / GND へ |
| 電圧スピーカ（ブザー） | GP1 | 圧電スピーカ等を直接接続 |
| LED | GP2 | 電流制限抵抗（約 330 Ω）を介して接続 |
| タクトスイッチ | GP3 | もう一方の端を GND へ接続。プルアップは内蔵設定で有効 |

## 使用方法

### タイマー時間の設定

ボリュームの回転位置でタイマー時間を 5 段階から選択します。

| ボリューム位置 | タイマー時間 |
|---------------|-------------|
| 0 〜 20 % | 1 分 |
| 20 〜 40 % | 2 分 |
| 40 〜 60 % | 3 分 |
| 60 〜 80 % | 4 分 |
| 80 〜 100 % | 5 分 |

### 操作手順

1. ボリュームで計測したい時間に合わせる。
2. スイッチを **短押し（1 秒未満）** するとタイマーが開始します。
   - LED が点灯し、計測中は LED が 1 秒ごとに交互点滅します。
3. スイッチを **長押し（1 秒以上）** すると設定時間確認モードになります。
   - LED が設定した分数の回数点滅して現在の設定を確認できます。
4. タイマー計測中にスイッチを押すとタイマーをキャンセルできます。
   - LED が約 3 秒間高速点滅してキャンセルを通知します。
5. タイマーが完了すると、ブザーから完了メロディが流れます。

## 内部の動き

```
[起動・初期化]
      │
      ▼
[LED 消灯、スイッチ押下待ち]
      │
      │ スイッチ押下
      ▼
[LED 点灯、ADC でボリューム読み取り → タイマー時間決定]
      │
      ├─── 1 秒以内に離す ──→ [タイマー開始]
      │                              │
      │                              │ 1 秒ごとにカウント
      │                              │ LED 交互点滅
      │                              │
      │                   ┌─── 途中スイッチ押下
      │                   │         │
      │                   │         ▼
      │                   │    [タイマーキャンセル]
      │                   │    LED 高速点滅（約 3 秒）
      │                   │         │
      │                   │         └──────────────────┐
      │                   │                             │
      │                   └── タイマー完了              │
      │                              │                  │
      │                              ▼                  │
      │                       [完了メロディ再生]        │
      │                       （ブザー音楽出力）        │
      │                              │                  │
      └─── 1 秒以上押し続ける        │                  │
                  │                  │                  │
                  ▼                  │                  │
      [設定時間確認モード]           │                  │
       LED を設定分数回点滅          │                  │
                  │                  │                  │
                  └──────────────────┴──────────────────┘
                                     │
                                     ▼
                              [最初に戻る]
```

### タイマー計時の仕組み

- クロック 8 MHz、プリスケーラ 1:64 → TMR0 は 32 µs ごとにカウントアップ
- TMR0 が 250 になるまでのループ（約 8 ms）を 125 回繰り返すことで **1 秒** を計測
- タイマー時間（秒数）分だけ上記 1 秒ループを繰り返す

### ブザー音楽の切り替え

`src/main.c` 内の `#define` を変更することで完了メロディを切り替え可能です。

| マクロ名 | 楽曲 |
|---------|------|
| `RAMEN` | ラーメン完成！歓喜のチャルメラ（デフォルト） |
| `SEIJA` | 聖者の行進 |
| `RADETUKI` | ラデツキー行進曲 |
| `TENGOKU` | 天国と地獄（地獄のギャロップ） |
| `MENUETTO` | メヌエット（ト長調） |
| `ORG_CRYSTAL_BREEZE` | オリジナル曲：Crystal Breeze |
